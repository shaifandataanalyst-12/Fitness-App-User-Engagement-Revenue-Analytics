# Fitness App User Engagement & Revenue Analysis

## Project Overview
This project analyzes user engagement and subscription data for a fitness application to understand user behavior, revenue performance, and churn patterns.

## Problem Statement
Fitness applications rely heavily on user engagement and subscription revenue. The objective of this project is to analyze user activity and identify patterns affecting user retention and revenue generation.

## Tools Used
- Power BI
- DAX
- Power Query

## Dataset
Fitness app dataset containing:
- User ID
- Subscription Type
- Workout Type
- Login Frequency
- Country
- Revenue

## Analysis Performed
- Data cleaning and transformation
- User segmentation analysis
- Revenue performance analysis
- Churn and retention analysis
- Made 15+ DAX measures for KPI analysis
- Dashboard development

## Key Insights
- The dataset contained **1,500+ users generating $379K in revenue**.
- Approximately **801 users churned**, indicating a high churn rate.
- **Cardio workouts were the most popular activity among users**.
- Paid subscription users showed higher engagement levels compared to free users.
